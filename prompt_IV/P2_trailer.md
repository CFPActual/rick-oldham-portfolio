# GreenCity Urban Tree Canopy Monitor

## Problem

Urban tree canopy provides environmental, public health, and climate benefits, yet many cities lack current and affordable canopy inventory data. Traditional tree surveys are often costly, labor-intensive, and quickly become outdated, making it difficult for municipalities to identify coverage gaps and support urban planning decisions.

## Approach

GreenCity combines aerial imagery, computer vision, and geospatial analysis to identify and monitor urban tree canopy at scale. The project incorporated image quality assessment, automated image enhancement, and multiple tree detection approaches to improve canopy visibility and detection performance using publicly available imagery.

## Results

The system successfully detected and localized thousands of trees across Pasadena, California using high-resolution aerial imagery. Image enhancement workflows improved model readiness, while computer vision models demonstrated strong recall and practical canopy monitoring capability. The project established a scalable workflow for generating actionable urban canopy data from remotely sensed imagery.

## Technologies

Python, TensorFlow, OpenCV, DeepForest, Google Earth Engine, GeoPandas, Rasterio, NumPy, Pandas

## Key Topics

Computer Vision, Object Detection, Geospatial Analytics, Remote Sensing, Urban Forestry, Environmental Data Science, Machine Learning
